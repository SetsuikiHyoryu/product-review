# 商品点评

## 技术栈

| 服务         | 技术                 |
| ------------ | -------------------- |
| 客户端运行时 | 浏览器               |
| 服务端运行时 | Bun                  |
| 服务端框架   | Elysia               |
| 渲染         | TSX 模版的服务端渲染 |
| 数据库       | SQLite               |
| 数据库访问   | bun:sqlite           |

## 设计

- 首页以 grid 的形式展示分类按钮，最后一项是新增分类。
  - 分类先不用独立表，用 `products.category TEXT` 起步。  
    首页查询 `SELECT DISTINCT category`。
- 点击分类按钮后进入对应分类的商品点评列表，表现形式为行式卡片 (list row card)。
  - 左图（预览一张图，点开后可以显示所有图，预览图右下角标记共有多少图）右内容。
  - 一卡片占满一整行，高度不固定。
  - 展示以下信息：
    - 预览图（新建时可以直接存图片或外链地址）。
    - 点评。
    - 商品名。
    - 店名。
    - 修改按钮（点击后进入和新增商品相同部局的画面进行修改）。
- 数据模型：
  - products
    - id
    - category
    - name
    - shop
    - comment
    - image_path_or_url
    - created_at（暂时用不到，但将来可能用到。）
- 右下角全画面悬浮一个新增商品点评的按钮。
- 另建私有数据仓库上传数据库本体以实现多终端同步。
